# aonav

## Dependencies
For running aonav:
- Rust/cargo
  https://rustup.rs/

For running the evaluation:
- uv
  https://docs.astral.sh/uv/getting-started/installation/

For MaxInfoGainStrategy:
- Ganak v2.5.3
  https://github.com/meelgroup/ganak/releases/tag/release%2F2.5.3

For building Docker image:
- podman
  https://podman.io/
